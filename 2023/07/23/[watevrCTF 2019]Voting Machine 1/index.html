[watevrCTF 2019]Voting Machine 1

一、前言

写这道题时比较搞笑，被自己蠢到了。

二、题目

开启了NX保护，到ida中静态分析

栈溢出点很明显。因为我没有在ida中寻找后门函数，我用了ROP这个工具去查system和’/bin/sh‘

所以我以为这道题又是一道ret2libc，然后就按照ret2libc的思路去打，但是最后的溢出调用system函数时，栈顶一直不能保持16字节对齐，一直打不通，所以感觉自己写出错了，去看了下别的大佬的文章，发现居然还有个后门函数，那这样就很简单了。

四、总结
